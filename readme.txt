Projeto de Validação de Formulário
Este projeto consiste em criar uma ferramenta que valide um formulário web, garantindo que as informações preenchidas pelos usuários estejam corretas e completas. O objetivo é melhorar a experiência do usuário e garantir a precisão dos dados coletados.

Funcionalidades
As funcionalidades do projeto incluem:

Verificar se todos os campos obrigatórios foram preenchidos.
Validar os formatos de dados, como endereço de e-mail, número de telefone e CPF.
Verificar se os dados estão dentro dos limites especificados, como quantidade máxima de caracteres ou valor mínimo e máximo.
Identificar e informar erros ao usuário em tempo real.
Exibir mensagens de confirmação após a conclusão bem-sucedida do envio do formulário.
Tecnologias Utilizadas
As tecnologias utilizadas no projeto incluem:

HTML para a criação do formulário web.
CSS para o design e estilo da interface.
JavaScript para a validação dos dados e interação com o usuário.
Como utilizar
Para utilizar o projeto, basta abrir o arquivo HTML em um navegador web. O formulário será exibido e as validações serão executadas automaticamente após o envio. O usuário receberá mensagens de erro ou confirmação, conforme necessário.
